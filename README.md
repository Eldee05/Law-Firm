This project is a clean and scalable frontend for a legal services website. It focuses on:

Professional UI design
Responsive layout
Modern frontend architecture

The goal is to create a sleek and trustworthy interface suitable for legal brands.

Tech Stack
React (UI library)
Vite (fast build tool)
Tailwind CSS (utility-first styling)
TypeScript (type safety)

I built this to
Practice modern frontend development
Learn Tailwind + React integration[for large scalable projects]
Build real-world UI for professional services

Getting Started
1. Clone the repository
git clone https://github.com/Eldee05/Law-Firm
2. Navigate into the project
cd law-Firm
3. Install dependencies
npm install
4. Run development server
npm run dev
